# ASCII Art Generator

A simple Rust program that converts text into ASCII art.

## Usage

1.  Clone the repository.
2.  Navigate to the project directory.
3.  Run `cargo run -- <text>` where `<text>` is the text you want to convert to ASCII art.

## Example

```bash
cargo run -- Hello, World!
```

## Dependencies

*   `figlet_rs` crate.
