# System Info Fetcher

A simple Rust program to fetch and display basic system information.

## Features

*   Retrieves OS name and version.
*   Retrieves CPU model name.
*   Retrieves total RAM.

## Usage

1.  Clone the repository.
2.  Build the project using `cargo build --release`.
3.  Run the executable located in `target/release/system_info`.

## Dependencies

*   [sysinfo](https://crates.io/crates/sysinfo)
