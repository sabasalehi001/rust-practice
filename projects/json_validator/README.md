# json_validator

A simple command-line tool to validate JSON files.

## Usage

```bash
json_validator <path_to_json_file>
```

## Features

*   Validates JSON files against the JSON specification.
*   Provides error messages if the JSON is invalid.

## Installation

```bash
cargo install --path .
```
